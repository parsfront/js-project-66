مسیر طراحی وب

  • Html & Css

    یادگیری حرفه ای این دو زبان ابتدایی

  • Bootstrap

    یادگیری کامل این فریم وورک که مخصوص زبان css است

  • Css grid - materialize

    یادگیری این تکنولوژی ها خالی از لطف نیست

  • Javascript

    آموزش کامل زبان جاوا اسکریپت جز الزامات یک طراح وب حرفه ای است

  • React js

    کتابخانه ری اکت از بهترین کتابخانه های جاوا اسکریپت است که با آن میشود سنگل پیج اپلیکیشن ساخت

  • Vue js

    ویو جی اس یک فریم وورک جاوا اسکریپت است و از لحاظ کارایی به ری اکت شباهت دارد

  • Angular

    انگولار هم یک فریم وورک محبوب جاوا اسکریپتی است مانند ویو و ری اکت و هر سه کاربرد یکسانی دارند

  • پیش به سوی موفقیت

    تبریک شما در حال حاضر یک طراح حرفه ای وب هستید.
